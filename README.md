# Multi-Agent System (Wiki Agent + Weather Agent)

## Features
- Wiki Agent → Fetches 2-line summary from Wikipedia  
- Weather Agent → Fetches live weather using OpenWeatherMap  
- Controller routes queries automatically:
  - “Who…” / “What…” → WikiAgent  
  - “Weather in London / Tell me weather of Delhi” → WeatherAgent
